Coding to build my OS

Folders and Files:

I created all the required folders like the source folder, implementation folder etc. I created a boot folder as well as a header.asm file. This file would contain data that needs to be included in our operating system. It is to help understand that an operating system can be run on our computers. This is because the operating system isn’t the first thing that starts on our computer. The bootloader is the first thing to start on our computers. We need the bootloader because it functions to locate our operating system. Only upon finding it will it be able to start the OS. I used the multi boot2 specification which is supported by most bootloaders.

Code:

Files were made global in order to be easily accessible. I entered a code to print “OK” first. Linker file was for linking the operating system together. Iso folder will have iso file which is holding my operating system. The commands for making my operating system are in my Makefile. I created a variable to hold my x86_64 assembly source files as well as object files. Afterwards I built it again and then ran qemu to emulate the operating system. The output screen printed OK and my Operating System was running.
